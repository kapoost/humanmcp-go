Creative work vanishes into training sets, gets paraphrased, attributed wrong, or simply absorbed. humanMCP gives every piece you publish a cryptographic signature and a Bitcoin-anchored timestamp — an unforgeable, independently verifiable record of authorship that any agent can check before using your work.Prace twórcze znikają w zbiorach treningowych, są parafrazowane, błędnie przypisywane albo po prostu wchłaniane. humanMCP nadaje każdemu publikowanemu utworowi podpis kryptograficzny i znacznik czasu zakotwiczony w Bitcoinie — niezbywalny, niezależnie weryfikowalny dowód autorstwa, który każdy agent może sprawdzić przed użyciem Twojej pracy.

bitcoin timestamp — how to verifyznacznik bitcoin — jak weryfikować

Every piece carries an OpenTimestamps proof anchored in the Bitcoin blockchain. You can verify any piece independently — no trust in this server required.Każdy utwór niesie dowód OpenTimestamps zakotwiczony w blockchainie Bitcoin. Możesz zweryfikować dowolny utwór niezależnie — nie wymaga zaufania do tego serwera.

Ⅰ
get the proofpobierz dowód
Ask any agent: "get_certificate piosenki" — the ots_proof field is a base64 OTS stub.Zapytaj dowolnego agenta: "get_certificate piosenki" — pole ots_proof to stub OTS w base64.
Ⅱ
decode itzdekoduj
echo "{ots_proof}" | base64 -d > piece.otsAfter ~1hr use the agent's upgrade_timestamp tool to get the full proof.Po ~1h użyj narzędzia agenta upgrade_timestamp, by pobrać pełny dowód.
Ⅲ
verify offlineweryfikuj offline
pip install opentimestamps-client ots verify piece.otsReturns the Bitcoin block height and timestamp. No account needed.Zwraca wysokość bloku Bitcoin i znacznik czasu. Nie wymaga konta.
