bitcoin timestamp — how to verifyznacznik bitcoin — jak weryfikować

Every piece carries an OpenTimestamps proof anchored in the Bitcoin blockchain. Verify independently — no trust in any server required.Każdy utwór niesie dowód OpenTimestamps zakotwiczony w blockchainie Bitcoin. Weryfikuj niezależnie — nie wymaga zaufania do żadnego serwera.

Ⅰ
get the proofpobierz dowód
Ask any agent: “get_certificate piosenki” — the ots_proof field is a base64 OTS stub.Zapytaj dowolnego agenta: „get_certificate piosenki” — pole ots_proof to stub OTS w base64.
Ⅱ
decode itzdekoduj
echo "{ots_proof}" | base64 -d > piece.otsAfter ~1hr use upgrade_timestamp to get the full proof.Po ~1h użyj upgrade_timestamp, by pobrać pełny dowód.
Ⅲ
verify offlineweryfikuj offline
pip install opentimestamps-client ots verify piece.otsReturns Bitcoin block height and timestamp.Zwraca wysokość bloku Bitcoin i znacznik czasu.
